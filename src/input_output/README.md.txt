# 📥 Input / Output (Java)

This module covers all **Input/Output-based problems in Java**, starting from basic console operations to advanced file handling and custom implementations.

---

## 🎯 Objective

* Master console input/output handling
* Work with files (text, CSV, JSON)
* Understand formatting and parsing
* Build strong foundational skills for real-world applications

---

## 📚 Problems Covered

---

### 🔹 Basic

1. Read integer, string from console
2. Print multiplication table
3. Print star triangle pattern
4. Print pyramid pattern
5. Print diamond pattern
6. Read/write text file
7. Sum of N numbers from input
8. Reverse input string
9. Count words in a sentence
10. Print ASCII value of character

---

### 🔸 Medium

1. Print Pascal’s triangle
2. Print calendar for given month/year
3. Read CSV file and parse data
4. Print number pyramid pattern
5. Print Floyd’s triangle
6. Print hollow square pattern
7. Print butterfly pattern
8. Print spiral matrix
9. Print checkerboard pattern
10. Print Pascal’s triangle using recursion

---

### 🔺 Hard

1. Implement custom scanner class
2. Implement formatted output like printf
3. Read JSON file and parse manually
4. Print Sudoku board from input
5. Print chessboard pattern
6. Print diamond with numbers
7. Print concentric square pattern
8. Print zig-zag matrix
9. Print Pascal’s triangle with memoization
10. Implement custom file reader with buffer

---

## 🧠 Key Concepts Covered

* Console input using `Scanner`
* Handling different data types
* Buffer handling issues (`nextInt()` vs `nextLine()`)
* File handling (read/write operations)
* Parsing structured data (CSV, JSON)
* Output formatting
* Pattern printing using loops

---

## ⚠️ Common Pitfalls

* Input skipping due to newline buffer
* Incorrect parsing of mixed data types
* Improper file handling (not closing resources)
* Incorrect loop structure in pattern problems

By completing this module, you will:
